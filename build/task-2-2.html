Решение задания 2 - Таблица на странице

Проблема заключается в стилях контейнера .block

.blog{ position: relative; margin: 0 auto; padding: 20px 15px; width: 990px; max-width: 95%; background-color: #fff; }

Из-за того, что максимальная ширина задана в процентах, а внутренний отступ - в пикселях, при уменьшении ширины окна в некоторый момент контейнер становится шире вьюпорта.

Для решения необходимо установить размер отступа в процентах: .blog{ position: relative; margin: 0 auto; padding: 20px 15px; padding: 20px 2.5%; width: 990px; max-width: 95%; background-color: #fff;

либо учитывать отступы в ширине контейнера: .blog{ position: relative; margin: 0 auto; padding: 20px 15px; width: 990px; max-width: 95%; max-width: 100%; box-sizing: border-box; background-color: #fff;
